# musicplayer.HTML5
Music Player HTML5, CSS &amp; JavaScript for Android Tablet &amp; Android Car.

How to open :
* On Android OS, install 'Simple HTTP Server' from Google Play, create htdocs folder on Download of internal storage, extract at htdocs, open your local server musicplayer.HTML5 using your Android browser.
* Work perfectly on Windows 7/10/11 using browser Chrome, Firefox, Edge by open index.html file or Open with.
* Use 'Simple HTTP Server' or 'AwebServer' on Android OS:Operating System.
